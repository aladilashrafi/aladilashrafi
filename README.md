## 🔌 Featured Project — Bangla Track

> **Courier Integration Platform for Bangladesh E-commerce**

Founded and built by me — a production-grade platform connecting WooCommerce stores with Bangladesh's top courier services.

| Feature | Details |
|---|---|
| 📦 Couriers | Steadfast · Pathao · RedX |
| 🏷️ Labels | PDF label generation via DOMPDF |
| 🔔 Tracking | Real-time webhook-based order tracking |
| 📋 Bulk Booking | Mass parcel booking from WooCommerce orders |
| 🔐 Licensing | Free / Starter / Pro (self-hosted update system) |
| ✅ Compatibility | HPOS-compatible · Nonce-verified AJAX handlers |
